# Seagrass Species Identification

Aplikasi identifikasi spesies lamun pada citra bawah air menggunakan EfficientDet-D0 dan YOLOv8, dikembangkan sebagai bagian dari penelitian skripsi di perairan Desa Pengudang, Pulau Bintan.
